# 熱浪增頻下台灣2050年夏季再生能源系統可靠度尾端風險
## 研究架構完整說明文件

> **研究期間：** 2050年7月（夏季尖峰）  
> **分析對象：** 台灣電力系統（≥70% 再生能源滲透率情境）  
> **核心問題：** 氣候暖化透過需求端（冷卻負荷上升）與供給端（太陽能效率下降）兩個機制，使2050年7月電力供需缺口的尾端風險惡化幅度為何？

---

## 一、研究問題與核心指標

### 1.1 研究問題

在台灣2050年再生能源滲透率達 ≥70% 的電力系統下，相較於當前氣候條件（2018–2022），2050年氣候情境（2041–2060）中7月每日最大供需缺口的額外增量有多大？其P95、P99尾端風險分別為何？主要不確定性來源為何？

### 1.2 核心指標

$$\text{DailyMaxGapIncrease}_t = \max_h \left\{ \left( \text{Load}_t^{\text{future}} - \text{Load}_t^{\text{baseline}} \right) + \left( \text{PV}_t^{\text{baseline}} - \text{PV}_t^{\text{future}} \right) \right\}$$

| 變數 | 定義 |
|------|------|
| $\text{Load}_t^{\text{baseline}}$ | 2018–2022年7月逐時平均負荷（MW） |
| $\text{PV}_t^{\text{baseline}}$ | 2050年裝置容量 × 2018–2022氣候下PV產量 |
| $\text{Load}_t^{\text{future}}$ | 2050年結構性負荷 + 氣候誘發增量 |
| $\text{PV}_t^{\text{future}}$ | 2050年裝置容量 × 2041–2060氣候下PV產量 |
| $h$ | 逐時時間步（0–23時） |

**詮釋：** 相較於2018–2022當前氣候條件，在2050年相同裝置容量下，單日最大供需缺口的額外增量（MW）。正值代表缺口惡化。

## 二、關鍵設計決策

| 項目 | 決定 | 說明 |
|------|------|------|
| Baseline定義 | 2018–2022歷史平均 | 以當前氣候為基準，量化氣候變遷增量效應 |
| 再生能源範疇 | 僅太陽能（PV） | 風電列為研究限制；熱浪期低風速效應為保守低估方向 |
| PV容量情境 | 40 / 60 / 80 GW 三情境 | 對應國發會2050淨零路徑目標區間 |
| 結構性負荷調整 | $K \sim \text{Uniform}(1.58,\ 2.11)$ | 納入蒙地卡羅不確定性分析（見第三節） |
| 熱浪識別 | 不做顯式事件篩選 | P95/P99尾端分析自然捕捉高溫極端日；TCCIP溫度分布右移隱含熱浪增頻效應 |
| 溫度代表 | ERA5-Land台灣面積平均（°C） | 與TCCIP合成氣候空間尺度一致 |
| 年固定效果 | 拔除 Year$_y$，保留線性 Trend$_t$ | 避免年虛擬變數無法外推至2050年的問題 |
| 模型驗證 | ERA5 1995–2014（實測）對照合成氣候同期 | 已備齊驗證所需資料 |

## 三、方法架構

### 3.1 需求端模型（Panel OLS + Bootstrap）

**訓練資料：** ERA5-Land 台灣面積平均氣溫（`averaged_t2m`，Kelvin → −273.15 → °C）× 台電逐時負荷（2018–2022年7月）

**迴歸模型：**

$$\text{Load}_t = \alpha + \sum_{h=1}^{23}\mu_h \cdot \text{Hour}_{h,t} + \sum_{h=0}^{23}\beta_h \cdot \left[\text{HotDegree}_t \times \mathbf{1}(\text{hour}_t = h)\right] + \sum_{w=1}^{6}\delta_w \cdot \text{Weekday}_{w,t} + \rho \cdot \text{Trend}_t + \varepsilon_t$$

| 變數 | 說明 |
|------|------|
| $\text{HotDegree}_t$ | 逐時氣溫超過21°C的累積度數（Cooling Degree Hour） |
| $\beta_h$ | 第 $h$ 小時每升溫1°C的負荷增量（24組獨立係數） |
| $\text{Hour}_{h,t}$ | 小時虛擬變數（控制基本日內負荷型態） |
| $\text{Weekday}_{w,t}$ | 星期虛擬變數（控制工作日/假日差異） |
| $\text{Trend}_t$ | 線性時間趨勢（捕捉長期結構性變化） |

**Bootstrap：** $B = 1{,}000$ 次，對 $\beta_h$ 係數取得不確定性分布（用於蒙地卡羅內層）

**2050年負荷投影：**

$$\text{Load}^{\text{future}}_{2050}(h,t) = \text{Load}^{\text{baseline}}(h) \times K + \sum_{h}\beta_h \cdot \Delta\text{HotDegree}(h,t)$$

- $K$：結構性總量調整係數（見3.3節）
- $\Delta\text{HotDegree}(h,t)$：2041–2060情境氣溫 vs. 2018–2022基準的CDD21差異

---

### 3.2 太陽能產量模型（PVWatts物理模型）

**資料欄位：** 合成氣候 `temp_air_c`（已為°C）+ `ssrd_wm2`（W/m²）

**模組溫度計算：**

$$T_{\text{module}} = T_{\text{air}} + \frac{\text{NOCT} - 20}{800} \times \text{SSRD}$$

**逐時PV產量：**

$$P_{\text{output}}(h,t) = \text{PV\_capacity} \times \eta \times \left[1 + \gamma_p \times (T_{\text{module}} - 25)\right]$$

| 參數 | 數值 | 說明 |
|------|------|------|
| NOCT | 45°C | 標準操作模組溫度（固定假設，內含標準風速） |
| $\gamma_p$ | −0.45%/°C | 多晶矽模組溫度係數 |
| $\eta$ | 0.77–0.85 | 系統整體效率（含逆變器損失） |
| PV容量 | 40 / 60 / 80 GW | 國發會目標區間三情境 |

> **重要：** PV^baseline 與 PV^future 皆採用2050年裝置容量，僅改變氣候輸入（T_air、SSRD），以正確隔離氣候效應而非容量擴張效應。

---

### 3.3 結構性負荷調整係數（$K$）

**來源與數值：**

| 資料來源 | 數值 | 計算基礎 |
|---------|------|---------|
| 國發會2050淨零路徑 — 低需求帶 | $K = 1.58$ | 2050年需求427.5 TWh ÷ 2020基準271.25 TWh |
| 國發會2050淨零路徑 — 高需求帶 | $K = 2.11$ | 2050年需求573.1 TWh ÷ 2020基準271.25 TWh |
| 國發院年均成長率佐證 | 2.0% ± 0.5% | 驗證區間合理性 |

**$K$ 隱含的需求驅動因素：**

| 正向驅動（拉升需求） | 負向驅動（抑制需求） |
|------------------|------------------|
| 工業直接電氣化（+40–41%） | 建築節能效率（抵銷潛在負荷約38%） |
| 電動車普及（2050目標90%滲透率） | 需求回應（DR）抵銷：1.72–9.67 GW |
| 智慧製造、資料中心擴張 | — |
| 空調冷卻需求持續成長 | — |

**蒙地卡羅設定：** $K \sim \text{Uniform}(1.58,\ 2.11)$，於中層迴圈抽樣。

### 3.4 三層巢狀蒙地卡羅

```
外層迴圈（200次）：5個氣候模型 × 2個SSP × 20年 July
│
├─ 中層迴圈（M次）：K ~ Uniform(1.58, 2.11) 抽樣
│  │
│  └─ 內層迴圈（B = 1,000次）：β_h 係數 bootstrap 抽樣
│       ↓
│     計算全7月每日（31天）DailyMaxGapIncrease
│       ↓
│     提取 均值、P95、P99
↓
彙整所有樣本分布
```

| MIROC6 | ✅ | ✅ |
| MPI-ESM1-2-HR | ✅ | ✅ |
| MRI-ESM2-0 | ✅ | ✅ |
| TaiESM1 | ✅ | ✅ |

### 3.5 Sobol不確定性分解

**三類不確定性來源：**

| 不確定性來源 | 對應隨機變數 | 描述 |
|------------|------------|------|
| 氣候不確定性 | 5模型 × 2 SSP | 模型間溫度、輻射的預測差異 |
| 統計不確定性 | $\beta_h$ bootstrap樣本 | 回歸係數估計誤差 |
| 結構性負荷不確定性 | $K \sim \text{Uniform}(1.58, 2.11)$ | 2050年電力需求結構不確定性 |

**輸出：** 一階Sobol指數（$S_i$）與全階Sobol指數（$S_{Ti}$），量化各來源對P95/P99的貢獻比例。

## 八、研究限制

1. **再生能源範疇**：本研究聚焦太陽能，未納入風電。熱浪期間高壓系統主導往往伴隨低風速，風電-熱浪共現效應可能使真實缺口更大，本研究為**保守低估**方向。

2. **熱浪事件識別**：以全7月尾端分布替代個別熱浪事件識別，熱浪增頻效應由TCCIP情境的溫度分布整體右移隱含體現，非直接對應CWA官方定義。

3. **負荷結構的均等縮放假設**：$K$ 對所有小時均等縮放基準負荷。實際2050年因大量PV滲透產生的「鴨子曲線」效應（夜尖峰前移），日內負荷結構將有所不同，此為簡化假設。

4. **電池儲能系統**：2050年大規模儲能的缺口緩衝效果未納入，本研究呈現的是**無儲能介入的保守缺口估計**。

5. **PV地理分布**：太陽能裝置容量以全台加總處理，未針對縣市級地理分布建模；不同地理位置的溫度差異可能影響加總溫度效應估計。
